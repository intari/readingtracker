раньше - это было:
---
***REMOVED*** это:

- часть старой идеи с In2.0
- отработка новых идей для Android-версии SysMonitor'а без риска поломать что-то важное
- и не надо слишком серьезно воспринимать...

Требования для сборки:
- настройка путей как на INARI5
- заменить facebook app/parse appid  на свои

Требования к железу:
- Android 4.4
- Аппаратная поддержка Bluetooth Smart (=Bluetooth Low Energy)
- Google Play Services (=на устройстве должен стоять Google Market)

Желательно для работы:
- активный NSLogger на 10.0.0.6:50000
- BLE HRM Sensor
- Sony SmartWatch 2
- Step detector/counter (у новых девайсов с 4.4 - есть)
